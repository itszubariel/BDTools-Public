Word Games

Wordle word generation and validation for Discord bots. Generate random 5-letter words and validate player guesses.

Word Games

Overview GET /random-word GET /validate-word

Overview

The Word Games API provides endpoints for building Wordle-style games in Discord bots. Generate random 5-letter words from a curated wordlist and validate player guesses against both the wordlist and an external dictionary. Both endpoints are public and require no authentication.

No auth required 
// Base URL for word game endpoints
https://api.bdtools.xyz
GET /random-word

Returns a random 5-letter word from a curated wordlist. Perfect for starting a new Wordle game. The wordlist contains common English words suitable for word-guessing games.

No auth required Rate limit: 30 requests per 10 seconds

Responses

200 OK 
{ "word": "apple" }
500 Internal Server Error — Empty Wordlist 
{ "error": "Word list is empty" }
500 Internal Server Error — File Read 
{
  "error": "Failed to load word list",
  "details": "ENOENT: no such file or directory"
}
GET /validate-word

Validates whether a 5-letter word is a valid English word. First checks the curated wordlist, then falls back to an external dictionary. Returns the validation result and the source of validation (wordlist or dictionary).

No auth required Rate limit: 60 requests per 10 seconds

Query Parameters

Parameter Type Description
word stringrequired The 5-letter word to validate. Must contain only letters (a-z, A-Z).
Example: /validate-word?word=apple

Responses

200 OK — Valid (Wordlist) 
{
  "valid": true,
  "word": "apple",
  "source": "wordlist"
}
200 OK — Valid (Dictionary) 
{
  "valid": true,
  "word": "zebra",
  "source": "dictionary"
}
200 OK — Invalid 
{
  "valid": false,
  "word": "xyzqw"
}
400 Bad Request — Missing Parameter 
{ "error": "Missing ?word= parameter" }
400 Bad Request — Wrong Length 
{ "error": "Word must be exactly 5 letters" }
500 Internal Server Error 
{
  "error": "Failed to reach dictionary",
  "details": "Network timeout"
}
502 Bad Gateway 
{
  "error": "Dictionary error",
  "status": 503
}
