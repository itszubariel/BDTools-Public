Before You Begin

To make the most of the BDTools Bot Guild List, follow these steps:

  1. Get your API key: You need a unique API key generated from our API Documentation to submit your bot’s server data securely.
  2. Integrate the API: Copy the code from the Example Field in the documentation and run it with the desired number of servers via your BDFD bot, Note: you need to use that code nothing else will work.
  3. Verify data submission: After submitting, check the website with your API key to see your servers listed. If a server doesn’t show up, either wait for a few more minutes or Contact Us.
  4. Keep your API key secure: Treat your API key like a password, don’t share it publicly to protect your bot’s data privacy.

Bot Guild List Information

Welcome to the BDTools Bot Guild List. This is the simple way to see where your bot is and what’s going on. Enter your API key, and you’ll instantly see your bot’s servers.

How It Works

Your bot sends us its server list using our secure API endpont /submit-server, we enrich the data with server icons, member counts, and presence counts, and then you can view it here.

Quick Start

Paste your API key above and you’ll get your full server list instantly. It’s simple: search, sort, and see stats like member counts, icons and description.

If you don’t see a new server, just resubmit your bot’s data.

The BDTools Bot Guild List is designed for developers who want complete insight into their bot’s servers, with detailed stats and advanced data enrichment.

How We Get the Data

Your bot sends its server list via our /submit-server endpoint. We then enhance this data using the Discord API to fetch additional details like member counts, server icons, and descriptions.

  • Server data includes: name, ID, owner ID, and invite link.
  • If no invite link is provided, some stats (like member count) may remain empty.

Security

  • Each user gets a unique JWT-based API key.
  • All requests are validated using Authorization headers and strict CORS.
  • Your data is private—no one can see your server list but you.

Using This Page

With your API key, you can fetch your bot’s full server list, filter servers, and view enriched data. If data is missing (like server icons), we fill it with placeholders for a clean UI.
