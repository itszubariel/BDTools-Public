About BDTools

Welcome to the BDTools Suite — a collection of browser-based utilities crafted for the Bot Designer for Discord (BDFD) community. No installs, no bots — just fast and efficient tools at your fingertips.

Whether you're an experienced scripter or just getting started, BDTools helps you:

  • Format and clean up messy command blocks
  • Escape tricky characters like ;, $, ], \
  • Generate templates for commands and event logic
  • Preview outputs before deploying them live

New features like Embed Builders, Performance Profilers, and more are added frequently to save you time and effort.

Have feedback or ideas? Contact us and help shape future updates.

About The API

The BDTools API is a secure and developer-friendly interface designed to let you connect your projects with our services. Whether you need to submit data, fetch server details, or manage integrations, this API is built to scale as we add more features and endpoints.

Authentication is done through unique API keys generated in your dashboard. Keep your key confidential, as it grants full access to your account's endpoints.

Our endpoints currently support server data submission and retrieval, but we have plans to add more functionalities soon. For detailed usage examples, rate limits, and upcoming updates, visit the API Documentation.
